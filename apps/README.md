# Benchmark Subject Applications

This directory contains local application subjects used by the benchmark.

The active RealWorld corpus runs against:

- `angular-realworld-example-app`
- `realworld`
- `vue3-realworld-example-app`

The app registry and adapters live in `src/apps/`. Maintained benchmark methodology documentation lives in `docs/`.

App folders may contain upstream README files or framework-specific notes. Treat those as subject-application documentation, not benchmark documentation.
